Об авторе

Здравствуйте. Меня зовут Юрий, я - Фронтенд-разработчик и автор News Analyzer. Данное приложение позволяет осуществлять поиск новостей по ключевым словам за последние семь дней, с сохранением результатов поиска. News Analyzer разработан с применением следующего стека технологий: JS, ES6, БЭМ, ООП, CSS, HTML, NPM, Webpack, GIT, также, применялись следующие библиотеки: Fetch API, GitHub API, News API, Swiper API.
